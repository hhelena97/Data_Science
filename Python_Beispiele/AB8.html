Maximalfluss und minimaler Schnitt¶

a)¶

Zielfunktion: $\sum f_{sv} = \sum f_{vt} \Rightarrow max (s,a) + (s,b) = (e,t) + (f,t)$,

Nebenbedingungen: $\forall e \in E: 1.f_{uv} \leq c_{uv} 0; 2.\sum f_{uv} - \sum f_{vw} = 0; 3. f(e) \geq 0$

Trennende Hyperebene¶

a)¶

$max \delta$ unter den Nebenbedingungen:

$1. x_{i}a + b + \delta \leq y_{i}$ fr die Punkte aus Gruppe 1, $a, b \in \mathbb{R}$

$2. x_{i}a + b - \delta \geq y_{i} \iff -x_{i}a - b + \delta \leq -y_{i}$ fr die Punkte aus Gruppe 2, $a, b \in \mathbb{R}$,

$3. \delta \geq 0$

gr1_x = [0.6, 1.0, 1.5, 2.5, 2.9, 3.0, 4.4, 5.6, 6.0, 7.5, 8.6, 10.6]                        #x Gruppe 1
gr1_y = [2.5, 0.9, 1.4, 2.5, 3.5, 1.1, 2.3, 1.6, 3.6, 2.4, 3.4, 2.5]                         #y Gruppe 1

gr2_x = [1.4, 2.6, 2.7, 3.5, 3.6, 4.1, 5.2, 5.5, 5.8, 7.2, 7.6, 9.6, 9.9, 11.1]              #x Gruppe 2
gr2_y = [-2.9, -2.5, -3.9, -1.5, -2.4, -3.6, -2.4, -1.4, -3.2, -1.2, -1.9, 0.6, -1.3, 1.7]   #y Gruppe 2

target_delta = [0, 0, -1]                                                                    #Zielfunktion: delta

gr1_mat = np.array([[1.0 for _ in range(3)]for _ in range(len(gr1_x))])
for i in range(len(gr1_x)): gr1_mat[i,0] *= gr1_x[i]                                         #Koeffizientenatrix fr NB1

gr2_mat = np.array([[-1.0, -1.0, 1.0]for _ in range(len(gr2_x))])
for i in range(len(gr2_x)): gr2_mat[i,0] *= gr2_x[i]                                         #Koeffizientenatrix fr NB2

d_constraint_ub_mat = np.concatenate((gr1_mat, gr2_mat), axis=0)                             #Koeffizientenatrix zusammengefgt
d_constraint_ub_vec = np.append(gr1_y, np.multiply(gr2_y, -1))                               #b-Vektor zusammengefgt

hyperplane = sp.optimize.linprog(c=target_delta, A_ub=d_constraint_ub_mat,                   #Lse min -delta
                                 b_ub=d_constraint_ub_vec, 
                                 bounds=[(None,None), (None,None), (0,None)])                #bounds fr a,b auf reelle Werte

print("Abstand delta: ", -hyperplane.fun)
print("Trennende Hyperebene: y =", hyperplane.x[0],"x",hyperplane.x[1])
